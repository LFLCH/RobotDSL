## Output

### Simulation
The simulation panel is opened after a successful interpretation of the editor's code.

Components :
- Stop button
- Play/pause button
- Slider

### Compilation
The compilation panel is opened after a successful compilation of the editor's code.

Components :
- Copy
- Save on disk
